## Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Setup Environment**
   ```bash
   cp .env.example .env
   ```
   Update the environment variables in `.env`

3. **Setup Database**
   ```bash
   npm run db:generate
   npm run db:push
   npm run db:seed
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   ```
