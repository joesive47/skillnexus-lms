# การแก้ไขปัญหา Video Player - SkillNexus LMS

## ปัญหาที่พบบ่อยและวิธีแก้ไข

### 1. ไม่สามารถเล่นวีดีโอได้

**สาเหตุที่เป็นไปได้:**
- ไม่มีการเชื่อมต่ออินเทอร์เน็ต
- YouTube ถูกบล็อกโดยไฟร์วอลล์หรือเครือข่าย
- วีดีโอถูกลบหรือเปลี่ยนเป็นส่วนตัว
- โปรแกรมป้องกันโฆษณา (Ad Blocker) รบกวน
- รหัสวีดีโอ YouTube ไม่ถูกต้อง

**วิธีแก้ไข:**
1. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
2. รีเฟรชหน้าเว็บ (F5 หรือ Ctrl+R)
3. ลองใช้เบราว์เซอร์อื่น (Chrome, Firefox, Edge)
4. ปิดโปรแกรมป้องกันโฆษณาชั่วคราว
5. ลองเปิดวีดีโอใน YouTube โดยตรง
6. ติดต่อผู้สอนหากปัญหายังคงอยู่

### 2. วีดีโอโหลดช้าหรือกระตุก

**วิธีแก้ไข:**
- ตรวจสอบความเร็วอินเทอร์เน็ต
- ปิดแอปพลิเคชันอื่นที่ใช้อินเทอร์เน็ต
- ลดคุณภาพวีดีโอ (หากมีตัวเลือก)
- ลองดูในช่วงเวลาที่เครือข่ายไม่ติดขัด

### 3. ข้อความแสดงข้อผิดพลาดต่างๆ

#### "ไม่พบรหัสวีดีโอ YouTube"
- ตรวจสอบลิงก์วีดีโอที่ให้มา
- แจ้งผู้สอนเพื่อตรวจสอบลิงก์

#### "เจ้าของวีดีโอไม่อนุญาตให้เล่นในเว็บไซต์นี้"
- วีดีโอถูกจำกัดการเล่นแบบฝัง (embed)
- ใช้ปุ่ม "เปิดใน YouTube" เพื่อดูวีดีโอ

#### "ไม่พบวีดีโอนี้"
- วีดีโออาจถูกลบหรือเปลี่ยนเป็นส่วนตัว
- ติดต่อผู้สอนเพื่อขอลิงก์ใหม่

### 4. ระบบ Anti-Skip ไม่ทำงาน

**สาเหตุ:**
- JavaScript ถูกปิดใช้งาน
- เบราว์เซอร์ไม่รองรับ

**วิธีแก้ไข:**
- เปิดใช้งาน JavaScript ในเบราว์เซอร์
- อัปเดตเบราว์เซอร์เป็นเวอร์ชันล่าสุด
- ใช้เบราว์เซอร์ที่รองรับ (Chrome, Firefox, Safari, Edge)

## การตรวจสอบระบบ

### ตรวจสอบการเชื่อมต่อ
1. เปิด https://www.youtube.com ในแท็บใหม่
2. หากเปิดได้ แสดงว่าการเชื่อมต่อปกติ
3. หากเปิดไม่ได้ ตรวจสอบการตั้งค่าเครือข่าย

### ตรวจสอบเบราว์เซอร์
- **Chrome**: เวอร์ชัน 90 ขึ้นไป
- **Firefox**: เวอร์ชัน 88 ขึ้นไป  
- **Safari**: เวอร์ชัน 14 ขึ้นไป
- **Edge**: เวอร์ชัน 90 ขึ้นไป

### ตรวจสอบ JavaScript
1. กด F12 เพื่อเปิด Developer Tools
2. ไปที่แท็บ Console
3. หากมีข้อผิดพลาดสีแดง ให้จับภาพหน้าจอส่งให้ผู้ดูแลระบบ

## การติดต่อขอความช่วยเหลือ

เมื่อติดต่อขอความช่วยเหลือ กรุณาแจ้งข้อมูลต่อไปนี้:

1. **ข้อความแสดงข้อผิดพลาด** (ถ้ามี)
2. **เบราว์เซอร์และเวอร์ชัน** (เช่น Chrome 120.0)
3. **ระบบปฏิบัติการ** (Windows, Mac, iOS, Android)
4. **ลิงก์วีดีโอที่มีปัญหา**
5. **ขั้นตอนที่ทำก่อนเกิดปัญหา**

## คุณสมบัติของระบบ Video Player

### Anti-Skip Technology
- ป้องกันการข้ามวีดีโอ
- ตรวจสอบความคืบหน้าแบบเรียลไทม์
- บันทึกจุดที่ดูล่าสุด

### การรองรับ
- รองรับวีดีโอ YouTube เท่านั้น
- รองรับการเล่นแบบ Fullscreen
- แสดงความคืบหน้าการดู
- ระบบ Retry อัตโนมัติ

### ข้อจำกัด
- ต้องมีการเชื่อมต่ออินเทอร์เน็ต
- ไม่รองรับวีดีโอที่ถูกจำกัดการเล่นแบบฝัง
- ไม่รองรับวีดีโอส่วนตัวหรือที่ถูกลบ
